# Home — `/`

Inherits MASTER. Calm, useful, not a marketing landing page.

## Sections (in order)
1. **Header** — display serif heading "What are you working on?" + supporting line "Start with
   something you already have. Harbor will help you make it clear." A quiet "Private beta" tag.
2. **Start new work / Universal intake** — one card. A plain-language sentence input (the only
   required field), an attach row (Paste text · Upload files · Add a link · Start from an idea),
   and one primary button "Continue". Below it, the inferred intent appears as a quiet line
   ("This looks like meeting preparation — you can change it") rather than a form.
3. **Most-used services** — a restrained grid of service cards grouped Create / Intelligence /
   Action. Each card: name, one-line purpose, small Lucide icon. No counts, no fake stats.
4. **Recent work** — only real local artifacts from IndexedDB. Empty state: "Your saved work
   will appear here. Nothing leaves this browser unless you export it." Never fake activity.
5. **Privacy reminder** — compact, links to `/privacy`.

## Notes
- Hero serif only here and in slide statements. Everything else sans.
- The intake card is the visual anchor; generous whitespace around it. No giant chat box —
  it's a labelled input with helper text, not a chat surface.
- Demo examples are allowed but always carry a visible "Demo example" tag.
